# Bounty Budget — Legal

This repository hosts the public legal documents for the **Bounty Budget** iOS app.

## Documents
- **Privacy Policy:** `/privacy/`
- **Terms of Use:** `/terms/`

These pages are published via **GitHub Pages** and are intended to be linked from:
- App Store Connect (Privacy Policy URL)
- The in-app “Legal” section

## About Bounty Budget
Bounty Budget is a personal finance and budgeting app that helps users track income, expenses, and financial goals. Users enter transactions manually (no bank sync). Optional AI features can generate insights and help parse voice-entered transactions.

## Publishing (GitHub Pages)
1. Go to **Settings → Pages**
2. Set **Source** to “Deploy from a branch”
3. Choose **Branch:** `main` and **Folder:** `/ (root)`
4. Save and wait for the Pages URL to appear

Once enabled, documents will be available at:
- `https://antonvoropaievgithub.io/<repo>/privacy/`
- `https://antonvoropaiev.github.io/<repo>/terms/`
